Demonstration

The deployed application demonstrates the complete workflow: - User
Registration and Login - Cab Search - Ride Booking - Booking History -
Driver Dashboard - Admin Dashboard

Clone the repository, install dependencies, configure environment
variables, and run the frontend and backend servers to explore the
project locally.
